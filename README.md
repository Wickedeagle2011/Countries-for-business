Hey there! This is a single-file, highly modular Python tool designed to pull live financial data from Yahoo Finance (yfinance) and break down the macroeconomic health of major global economies (the G20/G50).
I built this project to learn how to translate raw financial data streams into clean, actionable quant metrics without dealing with heavy database setups.
It uses a Data Mapping Engine Strategy to link real-time assets (market indexes, sovereign bonds, currency pairs) to static national profiles like corporate tax incentives and growth hubs.
